# KnowledgeBase(个人知识库)


## 简介

用来存放个人笔记和开发中积累的代码片段，作为一个备份和同步的云端。


## 目录结构

```
KnowledgeBase
  ├── .gitignore
  ├── README
  |
  ├── CodeSnippet
  │     └── db.json
  |
  └── NoteBook
        ├── Android
        ├── Python
        ├── Windows
        └── ...
```

## 环境
`db.json`文件使用massCode软件加载，`xxx.md`文件使用任意markdown软件可预览
